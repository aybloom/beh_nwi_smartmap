# Smartmap Walk Index
codefiles for building smartmap database and adding data

Work from BEH_SmartMap_Index.Rmd file to: 
  (1) Check the contents of the beh_nwi_smartmap database and existing raster files, and
  (2) Build out additional states/years in the database and in raster files, 
 
To generate values (including intersection density, business density, population density and composite walk score) for specific lat/lon points, see the get_walkscore_for_points.Rmd file for instructions.
